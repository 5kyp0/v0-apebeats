✅ What to Check

  • Title displays correctly and is under 60 characters
  • Description displays correctly and is under 160 characters
  • Image loads and displays properly (1200x630 pixels recommended)
  • No broken images or missing content
  • Preview looks good on mobile and desktop
  • Consistent branding across all platforms

🔧 Common Issues & Solutions

  • Images not loading: Check image URLs are accessible and HTTPS
  • Wrong title/description: Check meta tags in page source
  • Image too small/large: Ensure image is 1200x630 pixels
  • Caching issues: Use debugger tools to refresh cache
  • Mixed content: Ensure all URLs use HTTPS
